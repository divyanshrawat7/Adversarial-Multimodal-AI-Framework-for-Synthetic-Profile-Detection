
# Adversarial Multimodal AI Framework for Synthetic Profile Detection

An advanced multimodal AI framework for detecting AI-generated and synthetic online profiles using:

- Computer Vision
- Transformers
- Behavioral Analysis
- Multimodal Learning
- Adversarial AI

## Features

- Fake profile image detection
- Transformer-based bio analysis
- Behavioral anomaly detection
- Multimodal fusion
- Explainable AI dashboard
- Streamlit deployment

## Tech Stack

- PyTorch
- Transformers
- Streamlit
- OpenCV
- scikit-learn
- Google Colab
- Hugging Face Spaces
